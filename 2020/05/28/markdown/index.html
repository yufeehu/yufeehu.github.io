Markdown语法详解

#(空格)+标题名字 —— 几个#就是几级标题

标题

三级标题

四级标题

字体

Hello,World! ** ——文本左右两边放就是加粗

Hello,World! ——文本左右两边放*就是斜体

Hello,World! ——文本左右两边放***就是斜体且加粗

Hello,World! ——文本左右两边放~~就是划去

引用

我买几个橘子去。你就在此地，不要走动。——引用： >(空格)+文本

分割线

—或者***

图片

)——!+[描述]+(图片路径)

超链接

列表

  3. C

无序列表：-(空格)

  • A

  • C

快捷键
